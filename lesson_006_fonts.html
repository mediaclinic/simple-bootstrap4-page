Google Fonts

How to change your font?

Google Fonts

Go to fonts.google.com and select a font for your site.

  1. Press red plus sign on font
  2. Check the font info at the black box from bottom of the page
  3. Customize the font with weights you need
  4. Copy the Embed code, example: 
    <link href="https://fonts.googleapis.com/css?family=Open+Sans" rel="stylesheet">
  5. Paste it inside your html-document before link to custom.css (inside head-tag)
  6. Change the font inside your custom.css. Example below:

Usage in CSS


body {
	font-family: 'Open Sans', -apple-system, BlinkMacSystemFont, "Segoe UI", "Roboto", "Oxygen", "Ubuntu", "Cantarell", "Fira Sans", "Droid Sans", "Helvetica Neue", sans-serif;
}

Change weight


h1, h2, h3, .h1, .h2, .h3 {
	font-weight: 600;
}
