Keycommands

Boost your productivity with these basic keycommands.

Windows

Absolute basics:

Indent code right: Tab
Indent code left: shift + Tab
Save: ctrl + s
Copy: ctrl + c
Paste: ctrl + v

{ and }: alt gr + 7 and alt gr + 0
|: alt gr + <

Good to know:

Change program: ctrl + Tab
Open new tab in Browser: ctrl + t
Close tab in Browser: alt + w

Mac

Absolute basics:

Indent code: Tab
Indent code left: shift + Tab
Save: cmd + s
Copy: cmd + c
Paste: cmd + v

{ and }: alt + shift + 7 and alt + shift + 0
|: alt + 7

Good to know:

Change program: cmd + Tab
Open new tab in Browser: cmd + t
Close tab in Browser: cmd + w
