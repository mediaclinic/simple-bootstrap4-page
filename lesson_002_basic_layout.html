Basic layout

How to create columns in Bootstrap 4?

Basic structure


<div class="container">
	<div class="row">
		<div class="col">
			First column ever.
		</div>
	</div>
</div>

Basic structure, with semantics and typography


<div class="container">
	<div class="row">
		<div class="col">
			<h1>First H1 heading ever</h1>
			<p>Some text inserted here with p-tag.</p>
		</div>
	</div>
</div>

Basic structure for 3 columns


<div class="container">
	<div class="row">
		<div class="col-md-4">
			First column ever.
		</div>
		<div class="col-md-4">
			Second column ever.
		</div>
		<div class="col-md-4">
			Third column ever.
		</div>
	</div>
</div>

Basic structure for 3 columns, h3 and p


<div class="container">
	<div class="row">
		<div class="col-md-4">
			<h3>First heading h3.</h3>
			<p>First column content here.</p>
		</div>
		<div class="col-md-4">
			<h3>Second heading h3.</h3>
			<p>Second column content here.</p>
		</div>
		<div class="col-md-4">
			<h3>Third heading h3.</h3>
			<p>Third content column here.</p>
		</div>
	</div>
</div>
