Cards with Modals

How to create card with modal?

Card


<div class="card">
	<div class="zoom-effect-container">
		<div class="card-image">
			<a href="#" data-toggle="modal" data-target="#example1"><img class="card-image-top" src="http://placekitten.com/g/480/480"></a>
		</div>
		<ul class="list-unstyled p-3">
			<li class="small font-weight-bold text-muted text-uppercase">Category</li>
			<li>Description</li>
		</ul>
	</div>
</div>

Modal

Insert this modal code and remember to change id to same, what you have in image. It's #example1 in this example. 


<div class="modal fade" id="example1" tabindex="-1" role="dialog" aria-labelledby="exampleImage" aria-hidden="true">
	<div class="modal-dialog modal-lg">
		<class="modal-content">
			<img class="img-fluid" src="http://placebear.com/g/640/360">
		</div>
	</div>
</div>
